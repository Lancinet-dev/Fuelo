# Fuelo ⛽

Fuelo est un SaaS moderne de gestion de stations-service permettant de gérer le stock carburant, les ventes et les alertes en temps réel.

## 🚀 Fonctionnalités V1

- 🔐 Authentification JWT
- ⛽ Gestion du stock carburant
- 💰 Enregistrement des ventes
- 🚨 Alertes automatiques de stock faible
- 📊 Statistiques dashboard
- 🧾 Historique des ventes
- 🏢 Architecture multi-station

---

## 🛠️ Stack Technique

### Backend
- Node.js
- Express.js
- PostgreSQL
- JWT Authentication

### Frontend
- React
- Vite
- Tailwind CSS
